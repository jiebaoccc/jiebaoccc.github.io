【id317】B. DS排序--希尔排序

Dec 12, 2024
2 minute read

题目描述

给出一个数据序列，使用希尔排序算法进行降序排序。

间隔gap使用序列长度循环除2直到1

输入

第一行输入t，表示有t个测试示例 第二行输入n，表示第一个示例有n个数据(n>1) 第三行输入n个数据，都是正整数，数据之间用空格隔开 以此类推

输出

对每组测试数据，输出每趟排序结果。不同组测试数据间用空行分隔。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例

2
6
111 22 6 444 333 55
8
77 555 33 1 444 77 666 2222

输出样例

444 333 55 111 22 6
444 333 111 55 22 6

444 555 666 2222 77 77 33 1
666 2222 444 555 77 77 33 1
2222 666 555 444 77 77 33 1

解题思路

希尔排序，就是依次减小gap，然后每次在间隔为gap的序列中进行排序

gap的初始值为 len/2，然后每次 gap /= 2，直到 gap = 0 停止

代码思路即，将间隔为gap的数存入一个容器，对该容器进行排序，然后再将排序号的数依次存入对应的位置，如此循环

具体代码

#include<bits/stdc++.h>
using namespace std;

bool cmp(int a, int b) {
	return a > b;
}

int main() {
	int t;
	cin >> t;
	while (t--) {
		int n; cin >> n;
		vector<int> num(n);
		for (int i = 0; i < n; i++)
			cin >> num[i];

		int gap = n / 2;
		while (gap >= 1) {
			for (int i = 0; i + gap < n; i++) {
				vector<int> tem;
				for (int j = i; j < n; j += gap) {
					tem.push_back(num[j]);
				}
				sort(tem.begin(),tem.end(),cmp);
				for (int j = i,k=0; j < n; j += gap,k++) {
					num[j] = tem[k];
				}
			}
			gap /= 2;

			for (int j = 0; j < n; j++) {
				cout << num[j];
				if (j < n - 1)
					cout << " ";
				else cout << endl;
			}
		}
		cout << endl;
	}
	return 0;
}

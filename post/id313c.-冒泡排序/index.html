【id313】C. 冒泡排序

Dec 12, 2024
1 minute read

题目描述

给定一个包含从0到n-1各一次的数组，若使用冒泡排序将其排为升序，问其中需要进行多少次交换

输入

测试数据有多组，

每组由两行组成：第一行包含正整数n（n <= 5000）; 下一行包含从0到n-1的n个整数的序列。

输出

对于每组测试数据，

输出交换次数

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中

输入样例

10
1 3 6 9 0 8 5 7 4 2

输出样例

22

解题思路

因为本题没有输入测试组数，需要用到语句while(cin>>n)

具体代码

#include<bits/stdc++.h>
using namespace std;

int main() {
	int n; 
	while (cin>>n) {
		vector<int> num(n);
		for (int i = 0; i < n; i++)
			cin >> num[i];

		int ans = 0;
		for (int i = 1; i < n; i++) {
			int tag = 1;
			for (int j = 0; j < n - i; j++) {
				if (num[j] > num[j + 1])
				{
					swap(num[j], num[j + 1]);
					ans++;
					tag = 0;
				}
			}
			if (tag)
				break;
		}

		cout << ans << endl;
	}
	return 0;
}

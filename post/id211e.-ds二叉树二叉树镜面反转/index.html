【id211】E. DS二叉树—二叉树镜面反转

Oct 31, 2024
2 minute read

题目描述

假设二叉树用二叉链表存储，用含空子树遍历的先序序列结果创建。空子树用字符#表示

输入二叉树的先序序列，请你先创建二叉树，并对树做个镜面反转，再输出反转后的二叉树的先序遍历、中序遍历、后序遍历和层序遍历的序列。所谓镜面反转，是指将所有非叶结点的左右孩子对换。

–程序要求–

程序中不允许使用STL库等第三方对象或函数实现本题的要求

输入

测试次数t

每组测试数据是一个二叉树的先序遍历序列，#表示空树

输出

对每棵二叉树，输出镜面反转后的先序、中序、后序和层次遍历序列。如果空树，输出四个NULL（后面不加空格，每个NULL独自一行，中间没有空行）。如下：

NULL

NULL

NULL

NULL

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例

3
41#32###65##7##
AB#C##D##
AB##C##

输出样例

4 6 7 5 1 3 2 
7 6 5 4 3 2 1 
7 5 6 2 3 1 4 
4 6 1 7 5 3 2 
A D B C 
D A C B 
D C B A 
A D B C 
A C B 
C A B 
C B A 
A C B

具体代码

#include<bits/stdc++.h>
using namespace std;

class Tree
{
public:
	char data;
	Tree* left;
	Tree* right;
	Tree(char d) :data(d) {}
	Tree() {}
};

void Create(Tree*& t)
{
	char ch = getchar();
	if (ch == '#' || ch == '\n')
		return;

	t = new Tree(ch);
	Create(t->left);
	Create(t->right);
}

void Reverse(Tree*& t)
{
	if (!t)
		return;

	Tree* tem;
	tem = t->right;
	t->right = t->left;
	t->left = tem;

	Reverse(t->left);
	Reverse(t->right);
}

void Preorder(Tree* t)
{
	if (!t)
		return;
	cout << t->data << " ";
	Preorder(t->left);
	Preorder(t->right);
}

void Midorder(Tree* t)
{
	if (!t)
		return;
	Midorder(t->left);
	cout << t->data << " ";
	Midorder(t->right);
}

void Backorder(Tree* t)
{
	if (!t)
		return;
	Backorder(t->left);
	Backorder(t->right);
	cout << t->data << " ";
}

void Layerorder(Tree* t)
{
	Tree** q = new Tree * [1000] {NULL};
	q[0] = t;
	int n = 1;

	while (n)
	{
		Tree* tem = q[0];
		for (int i = 0; i <= n; i++)
		{
			q[i] = q[i + 1];
		}
		n--;

		if (!tem)
			continue;

		q[n] = tem->left;
		n++;
		q[n] = tem->right;
		n++;

		cout << tem->data << " ";
	}
}

int main()
{
	int t;
	cin >> t;
	while (t--)
	{
		getchar();
		Tree* tree = NULL;
		Create(tree);
		if (!tree)
		{
			for (int i = 0; i < 4; i++)
				cout << "NULL" << endl;
			continue;
		}
		Reverse(tree);

		Preorder(tree);
		cout << endl;
		Midorder(tree);
		cout << endl;
		Backorder(tree);
		cout << endl;
		Layerorder(tree);
		cout << endl;
	}
	return  0;
}

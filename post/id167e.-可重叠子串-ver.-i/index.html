【id167】E. 可重叠子串 (Ver. I)

Oct 11, 2024
1 minute read

题目描述

给定一个字符串（模式串）和一些待查找的字符串，求每个待查找字符串在模式串中出现的次数（可重叠）

输入

第一行输入t，表示有t组测试数据

每一组测试数据包含多行：

每一组的第一行包括一个字符串P，长度不超过105，且非空串

每一组的第二行包括一个整数N，代表待查找的字符串数量 (1 <= N <= 5)

每一组接下来的N行，每一行包括一个待查找的字符串，其长度不超过50，且非空串

输出

对于每组测试数据，

输出每个待查找字符串出现的次数，

具体输出见样例

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例1

2
aabbcc
3
aa
bb
cc
ababab
1
aba

输出样例1

aa:1
bb:1
cc:1
aba:2

解题思路

通过find函数找到该子串的起始位置，找到后计数加一，然后将该字符串从起始点到pos点的字符删除，接着继续寻找该子串的起始位置，直到

pos == string::npos，然后跳出循环，返回计数

具体代码

#include<iostream>
#include<string>
#include<algorithm>
using namespace std;

void NUM(string s,string str)
{
	int num = 0;
	while (1)
	{
		int pos = s.find(str);
		if (pos == string::npos)
			break;

		num++;
		s.erase(0, pos+1);
	}
	cout << str << ":" << num << endl;
}

int main()
{
	int t;
	cin >> t;
	while (t--)
	{
		string s;
		cin >> s;

		int n;
		cin >> n;

		while (n--)
		{
			string str;
			cin >> str;
			NUM(s, str);
		}
	}
	return 0;
}

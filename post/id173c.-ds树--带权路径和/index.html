【id173】C. DS树--带权路径和

Oct 31, 2024
1 minute read

题目描述

二叉树的创建使用含空树表示的先序遍历序列，计算一棵二叉树的带权路径总和，即求赫夫曼树的带权路径和。

已知一棵二叉树的叶子权值，该二叉树的带权路径和WPL等于叶子权值乘于根节点到叶子的分支数，然后求总和。

如下图中，叶子都用大写字母表示，权值对应为：A-7，B-6，C-2，D-3

树的带权路径和 = 71 + 62 + 23 + 33 = 34

输入

第一行输入一个整数t，表示有t个二叉树

第二行输入一棵二叉树的先序遍历结果，空树用字符‘0’表示，注意输入全是英文字母和0，其中大写字母表示叶子

第三行先输入n表示有n个叶子，接着输入n个数据表示n个叶子的权值，权值的顺序和前面输入的大写字母顺序对应

以此类推输入下一棵二叉树

输出

输出每一棵二叉树的带权路径和

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例

2
xA00tB00zC00D00
4 7 6 2 3
ab0C00D00
2 10 20

输出样例

34
40

解题思路

在先序生成二叉树时，记录对应的层数，当碰到大写字母时，将对应层数存入容器，然后计算

具体代码

#include<bits/stdc++.h>
using namespace std;

class Tree {
public:
	int data;
	Tree* left;
	Tree* right;
	Tree(char d) :data(d) {}
	Tree() {}
};

void Create(Tree* t,int h,vector<int>& ans)
{
	char ch = getchar();
	if (ch == '0' || ch == '\n')
		return;
	if (ch <= 'Z')
		ans.push_back(h);

	t = new Tree(ch) ;
	Create(t->left,h+1,ans);
	Create(t->right,h+1,ans);
}


int main()
{
	int tag;
	cin >> tag;
	getchar();
	while (tag--)
	{
		Tree* t = new Tree();
		int h=0;
		vector<int> ans;
		Create(t,h,ans);

		int sum = 0,n;
		cin >> n;
		getchar();
		for (int i = 0; i < ans.size(); i++)
		{
			int num;
			cin >> num;
			getchar();
			sum += ans[i] * num;
		}
		cout << sum << endl;
	}
	return 0;
}

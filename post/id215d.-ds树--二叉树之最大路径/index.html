【id215】D. DS树--二叉树之最大路径

Oct 31, 2024
1 minute read

题目描述

给定一颗二叉树的逻辑结构（先序遍历的结果，空树用字符‘0’表示，例如AB0C00D00），建立该二叉树的二叉链式存储结构

二叉树的每个结点都有一个权值，从根结点到每个叶子结点将形成一条路径，每条路径的权值等于路径上所有结点的权值和。编程求出二叉树的最大路径权值。如下图所示，共有4个叶子即有4条路径，

路径1权值=5 + 4 + 11 + 7 = 27路径2权值=5 + 4 + 11 + 2 = 22

路径3权值=5 + 8 + 13 = 26路径4权值=5 + 8 + 4 + 1 = 18

可计算出最大路径权值是27。

该树输入的先序遍历结果为ABCD00E000FG00H0I00，各结点权值为：

A-5，B-4，C-11，D-7，E-2，F-8，G-13，H-4，I-1

输入

第一行输入一个整数t，表示有t个测试数据

第二行输入一棵二叉树的先序遍历，每个结点用字母表示

第三行先输入n表示二叉树的结点数量，然后输入每个结点的权值，权值顺序与前面结点输入顺序对应

以此类推输入下一棵二叉树

输出

每行输出每棵二叉树的最大路径权值，如果最大路径权值有重复，只输出1个

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例

2
AB0C00D00
4 5 3 2 6
ABCD00E000FG00H0I00
9 5 4 11 7 2 8 13 4 1

输出样例

11
27

解题思路

先序遍历创建二叉树并赋予权重，然后返回左右子树的最大路权值

具体代码

#include<bits/stdc++.h>
using namespace std;

class Tree {
public:
	char data;
	int weight;
	Tree* left;
	Tree* right;

	Tree(char d) :data(d) {}
	Tree() {}
};

void Create(Tree*& t)
{
	char ch = getchar();
	if (ch == '0' || ch == '\n')
		return;

	t = new Tree(ch);
	Create(t->left);
	Create(t->right);
}

void Preorder(Tree* t,vector<int>& w)
{
	if (!t)
		return;

	t->weight = w[0];
	w.erase(w.begin());

	Preorder(t->left, w);
	Preorder(t->right, w);
}

int GetMax(Tree* t)
{
	if (!t)
		return 0;
	int w = t->weight;
	w += max(GetMax(t->left), GetMax(t->right));
	return w;
}

int main()
{
	int t;
	cin >> t;
	getchar();
	while (t--)
	{
		Tree* tree = new Tree();
		Create(tree);

		int n;
		cin >> n;
		
		vector<int> weight(n);
		for (auto& w : weight)
			cin >> w;
		getchar();

		Preorder(tree,weight);
		cout << GetMax(tree) << endl;
	}
	return 0;
}

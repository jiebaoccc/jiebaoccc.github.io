【id186】C. 树结构转换（先序转双亲）

Oct 26, 2024
2 minute read

题目描述

给出一棵二叉树的特定字符先序遍历结果（空子树用字符’#‘表示），构建该二叉树，并输出该二叉树的双亲表示法结果

双亲表示法的数组下标从0开始，根结点必定是在下标0元素，且根结点的双亲下标为-1，左右孩子按下标递增顺序排列,

结点下标是层次遍历顺序。

输入

第一个输入t，表示有t棵二叉树

接着t行，每行输入含特定字符的二叉树先序遍历序列

输出

共输出2t行

每棵二叉树输出两行，第一行输出各个结点的数值，第二行输出各结点的双亲下标

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例：

3
AB#C##D##
ABD##E##C##
AB##CDW###E#F##

输出样例：

A B D C
-1 0 0 1
A B C D E
-1 0 0 1 1
A B C D E W F
-1 0 0 2 2 3 4

解题思路：

首先创建树，根据先序遍历首先创建一棵树

然后对数进行层次遍历，因为是二叉树，所以当子树遍历数目达到两个时，其双亲下标则加一

而层次遍历是通过队列实现，首先将根节点入队，然后进入循环，每次循环出队一个节点，然后将其左右节点进行入队，出队时，如果为空节点，则将子树个数加一，然后continue

然后将出队的data存入容器，和当前双亲下标进行入队，最后输出

具体代码：

#include<bits/stdc++.h>
using namespace std;

class Tree
{
public:
	char data;
	Tree* left;
	Tree* right;
	Tree() {}
	Tree(char d) :data(d) {}
};

void create(Tree*& t,string& s)
{
	char ch = s[0];
	s.erase(0,1);
	if (ch == '#' || ch=='\0')
		return;

	t = new Tree(ch);
	create(t->left,s);

	if (s.empty())
		return;

	create(t->right,s);
}

void layer_order(Tree* t)
{
	queue<Tree*> q;
	vector<char> data;
	vector<int> parent;

	q.push(t);
	int tag = -1,num=1;

	while (!q.empty())
	{
		Tree* t = q.front();
		q.pop();

		if (!t)
		{
			num--;
			if (num == 0)
			{
				num = 2;
				tag++;
			}
			continue;
		}

		q.push(t->left);
		q.push(t->right);
		data.push_back(t->data);
		parent.push_back(tag);
		num--;
		if (num == 0)
		{
			num=2;
			tag++;
		}
	}

	for (int i = 0; i < data.size(); i++)
	{
		cout << data[i];
		if (i < data.size() - 1) cout << " ";
		else cout << endl;
	}

	for (int i = 0; i < parent.size(); i++)
	{
		cout << parent[i];
		if (i < parent.size() - 1) cout << " ";
		else cout << endl;
	}
}

int main()
{
	int t;
	cin >> t;
	while (t--)
	{
		string s;
		cin >> s;

		Tree* tr = new Tree();
		create(tr,s);
		layer_order(tr);
	}
	return 0;
}

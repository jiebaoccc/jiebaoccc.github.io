【id287】E. DS哈希查找--Trie树

Dec 06, 2024
2 minute read

题目描述

Trie树又称单词查找树，是一种树形结构，如下图所示。

它是一种哈希树的变种。典型应用是用于统计，排序和保存大量的字符串（但不仅限于字符串），所以经常被搜索引擎系统用于文本词频统计。它的优点是：利用字符串的公共前缀来节约存储空间，最大限度地减少无谓的字符串比较，查询效率比哈希表高。

输入的一组单词，创建Trie树。输入字符串，计算以该字符串为公共前缀的单词数。

（提示：树结点有26个指针，指向单词的下一字母结点。）

输入

测试数据有多组

每组测试数据格式为：

第一行：一行单词，单词全小写字母，且单词不会重复，单词的长度不超过10

第二行：测试公共前缀字符串数量t

后跟t行，每行一个字符串

输出

每组测试数据输出格式为：

第一行：创建的Trie树的层次遍历结果

第2~t+1行：对每行字符串，输出树中以该字符串为公共前缀的单词数。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例

abcd abd bcd efg hig
3
ab
bc
abcde

输出样例

abehbcficddggd
2
1
0

解题思路

首先创建一棵树，遍历输入字符串每个字母所对应的分支，如果对应的分支为空，则插入新节点，然后进入该节点，再遍历该字符串的第二个字母

寻找前缀即进入依次访问节点，如果节点为空，则直接返回false，反之继续深入

计算字母数量需要用到递归的方法，当所有分支都为空时，返回1，反之用ans统计每个分支中不同单词的数量

具体代码

#include<bits/stdc++.h>
using namespace std;

class Tree {
public:
	char data;
	vector<Tree*> branch;

	Tree(char data) {
		this->data = data;
		this->branch = vector<Tree*>(26, NULL);
	}
	Tree() { this->branch = vector<Tree*>(26, NULL); }
};

void createTree(Tree*& tree) {
	while (1) {
		string str;
		cin >> str;
		char ch;
		ch = getchar();

		Tree* t = tree;
		for (int i = 0; i < str.size(); i++) {
			if (t->branch[str[i] - 'a'] == NULL) {
				t->branch[str[i] - 'a'] = new Tree(str[i]);
			}
			t = t->branch[str[i] - 'a'];
		}

		if (ch == '\n')
			break;
	}
	return;
}

void LayerOutput(Tree* tree) {
	queue<Tree*> q;
	q.push(tree);

	while (!q.empty()) {
		Tree* t = q.front();
		q.pop();

		for (int i = 0; i < 26; i++)
		{
			if (t->branch[i] != NULL)
				q.push(t->branch[i]);
		}

		if (t->data != NULL)
			cout << t->data;
	}
	cout << endl;
}

int Calculate(Tree* tree) {
	int ans = 0;
	for (int i = 0; i < 26; i++) {
		if (tree->branch[i] != NULL) {
			ans += Calculate(tree->branch[i]);
		}
	}
	if (ans == 0)
		return 1;
	return ans;
}

int Find(Tree* tree, string str) {
	for (int i = 0; i < str.size(); i++) {
		if (tree->branch[str[i] - 'a'] != NULL)
			tree = tree->branch[str[i] - 'a'];
		else
			return 0;
	}

	return Calculate(tree);
}


int main() {
	Tree* tree = new Tree();
	createTree(tree);
	LayerOutput(tree);

	int t; cin >> t;
	while (t--) {
		string str;
		cin >> str;
		int val = Find(tree, str);
		cout << val << endl;
	}

	return 0;
}s

【id168】D. DS串应用—最长重复子串

Oct 11, 2024
1 minute read

题目描述

求串的最长重复子串长度（子串不重叠）。例如：abcaefabcabc的最长重复子串是串abca，长度为4。

输入

测试次数t

t个测试串

输出

对每个测试串，输出最长重复子串长度，若没有重复子串，输出-1.

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例1

3
abcaefabcabc
szu0123szu
szuabcefg

输出样例1

4
3
-1

解题思路

暴力解法，因为不能重叠，所以最长为 len/2 ，所以从 len/2 开始判断，设置第一层循环

第二层循环 ，j 从0开始，因为不能重叠，所以 j<len-2*i ， 当 j>=len-2*i时，两长度为 i 的子串已经重叠

定义两个子串，子串1 为s从j开始截取的长度为i的子串，子串2 为s从j+i之后的子串

如果在子串2中能找到子串1，则表示找到了最长重复子串，返回i

具体代码

#include<iostream>
#include<string>
#include<vector>

using namespace std;

int GetMaxLen(string s)
{
	int len = s.size();
	string s1, s2;

	for (int i = len / 2; i > 0; i--)
	{
		for (int j = 0; j < len - 2*i; j++)
		{
			s1 = s.substr(j, i);
			s2 = s.substr(j+i);
			if (s2.find(s1) != string::npos)
				return i;
		}
	}
	return -1;
}


int main()
{
	int t;
	cin >> t;
	while (t--)
	{
		string s;
		cin >> s;

		int max = GetMaxLen(s);
		cout << max << endl;
	}
	return 0;
}

【id382】A. DS排序--简单选择排序

Dec 19, 2024
2 minute read

题目描述

给出一个数据序列，使用简单选择排序算法进行升序排序。

输入

第一行输入t，表示有t个测试示例 第二行输入n，表示第一个示例有n个数据(n>1) 第三行输入n个数据，都是正整数，数据之间用空格隔开 以此类推

输出

对每组测试数据，输出每趟排序结果。不同组测试数据间用空行分隔。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

解题思路

选择排序，从最低位开始，依次选择未选择序列中的最小值与最低位进行交换

具体代码

#include<bits/stdc++.h>
using namespace std;

void simple_sort(vector<int> num,int n) {
	int i, j;
	for (i = 0; i < n - 1; i++) {
		int min = INT_MAX, min_index = 0;
		for (j = i; j < n; j++) {
			if (num[j] < min) {
				min = num[j];
				min_index = j;
			}
		}
		swap(num[i], num[min_index]);

		for (int i = 0; i < n; i++) {
			cout << num[i];
			if (i < n - 1) cout << " ";
			else cout << endl;
		}
	}
	cout << endl;
}

int main() {
	int t; cin >> t;
	while (t--) {
		int n; cin >> n;
		vector<int> num(n);
		for (int i = 0; i < n; i++) {
			cin >> num[i];
		}

		simple_sort(num,n);
	}
	return 0;
}

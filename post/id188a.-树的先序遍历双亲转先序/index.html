【id188】A. 树的先序遍历（双亲转先序）

Oct 26, 2024
1 minute read

题目描述

给出一棵树的双亲表示法结果，用一个二维数组表示，位置下标从0开始，如果双亲位置为-1则表示该结点为根结点

编写程序，输出该树的先根遍历结果。

输入

第一个输入t，表示有t棵树

接着每棵树输入3行：

第1行输入n，表示树有n个结点

第2行输入n个英文字母，表示每个树结点的数值

第3行输入n个整数，表示每个结点的双亲在数组的下标

以此类推输入下一棵树

输出

共输出t行，每行输出一棵树的先根遍历结果

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例：

2
7
A B C D E F G
-1 0 0 0 1 1 3
10
A B C D R E F G H K
4 4 4 0 -1 0 2 6 6 6

输出样例：

ABEFCDG
RADEBCFGHK

解题思路：

首先通过pair将data与val进行配对，每个节点都与其双亲节点的下标进行配对；通过两次循环，首先输入pair的第一个参数，data，接着输入第二个参数双亲下标，然后进入先根输出函数

函数中，遍历整个point，如果找到一个pair的第二位数等于函数中的双亲类，再次递归进入函数，将双亲下标替换成当前下标

具体代码：

#include<bits/stdc++.h>
using namespace std;

void preorder(pair<char, int>* point,int parent_index,int len)
{
	for (int i = 0; i < len; i++)
	{
		if (point[i].second == parent_index)
		{
			cout << point[i].first;
			preorder(point, i, len);
		}
	}

}

int main()
{
	int t;
	cin >> t;
	while (t--)
	{
		
		int n;
		cin >> n;
		pair<char, int>* point = new pair<char,int> [n];
		for (int i = 0; i < n; i++)
		{
			cin >> point[i].first;
		}
		for (int i = 0; i < n; i++)
			cin >> point[i].second;

		preorder(point,-1,n);
		cout << endl;
	}
	return 0;
}

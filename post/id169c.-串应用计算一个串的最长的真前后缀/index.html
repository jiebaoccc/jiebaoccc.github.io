【id169】C. 串应用- 计算一个串的最长的真前后缀

Oct 11, 2024
1 minute read

题目描述

给定一个串，如ABCDAB，则ABCDAB的真前缀有：{ A, AB,ABC, ABCD, ABCDA }ABCDAB的真后缀有：{ B, AB,DAB, CDAB, BCDAB } 因此，该串的真前缀和真后缀中最长的相等串为AB，我们称之为该串的“最长的真前后缀”。试实现一个函数string matched_Prefix_Postfix(string str)，得到输入串str的最长的真前后缀。若不存在最长的真前后缀则输出empty

输入

第1行：串的个数 n第2行到第n+1行：n个字符串

输出

n个最长的真前后缀，若不存在最长的真前后缀则输出empty。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例1

6
a
ab
abc
abcd
abcda
abcdab

输出样例1

empty
empty
empty
empty
a
ab

解题思路

通过KMP算法算出长度为 s.size()+1 的next数组，因为next数组每一位值也代表这一位之前的字符串的最长前后缀的长度，所以将字符串最后一位也进行判断，next数组的最后一位数字就是最长前后缀的长度

当没有前后缀时，next数组的最后一位数字为0

具体代码

#include<iostream>
#include<string>
#include<vector>
using namespace std;

int GetNext(string s)
{
	int i = 0, j = -1;
	int* next = new int[s.size()+1];
	next[0] = -1;
	while (i < s.size())
	{
		if (j == -1 || s[i] == s[j])
		{
			i++;
			j++;
			next[i] = j;
		}
		else j = next[j];
	}
	return next[s.size()];
}

int main()
{
	int t;
	cin >> t;
	while (t--)
	{
		string s;
		cin >> s;

		int pos = GetNext(s);
		if (pos == 0)
		{
			cout << "empty" << endl;
			continue;
		}

		for (int i = 0; i < pos; i++)
		{
			cout << s[i];
		}
		cout << endl;
	}
	return 0;
}

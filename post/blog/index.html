2.for循环的基础语法

1）基础语法

  • 格式：

    for 临时变量 in 待处理数据：
	循环满足条件是执行的指令

    待处理数据严格来说，称之为：序列类型

    序列类型指，其内容可以一个个依次取出的一种类型，包括：

    • 字符串
    • 列表
    • 元组
    • 等

  • 示例：

    # 定义字符串name
name = "jojo"
for i in name:
    print(i)

    # 输出结果
j
o
j
o

    将name的内容，依次取出赋予x临时变量

    可以称为遍历循环

  • 同while循环不同，for循环是无法定义循环条件的，只能从别处理的数据集中依次取出内容进行处理

    所以，理论上讲，Python中的for循环无法构建无限循环

2） range语句

  • 语法1：

    range（num）

    获取一个从0开始，到num结束的数字序列（不包含num本身）左闭右开

    如range（5）获得的数据是：[0,1,2,3,4]

  • 示例1：

    # range语法1 range(num)
for x in range(3):
    print(x)

    # 输出结果

1
2

  • 语法2：

    range（num1，num2）

    获得一个从num1开始到num2结束的数字序列（不包含num2本身）

    左闭右开

  • 示例2：

    # range 语法2 range(num1,num2)
for x in range(3,6):
    print(x)

    # 输出结果
3
4
5

  • 语法3：

    range（num1，num2，step）

    获得一个从num1开始，到num2结束的数字序列（不含num2本身）

    数字之间的步长，即间隔长度，以step为准（step默认为1）

    如，range（5,10,2）取得的数据是：[5,7,9]

  • 示例3：

    # range 语法3 range(num1,num2,step)
for x in range(3,10,2):
    print(x)

    # 输出结果
3
5
7
9

6.示例：银行服务

  • 代码：

    money = 500000
name = None

name = input("请输入你的名字：")

def query(show_header):
    if(show_header):
        print("------------查询余额------------")
    print(f"{name},您好，您的余额剩余：{money}元。")

def saving(num):
    global money
    money += num
    print("------------存款------------")
    print(f"{name},您好，您存款：{num}成功。")
    query(False)

def get_money(num):
    global money
    money -= num
    print("------------取款------------")
    print(f"{name},您好，您取款：{num}成功。")
    query(1)

def main():
    print("------------主菜单------------")
    print(f"请操作：")
    print("查询余额\t[输入1]")
    print("存款\t[输入2]")
    print("取款\t[输入3]")
    print("退出\t[输入4]")
    return input("请输入你的选择：")

# 设置无限循环，确保程序不退出
while True:
    keyboard_input = main()
    if keyboard_input == "1":
        query(1)
        continue
    elif keyboard_input == "2":
        num = int(input("请输入存款金额："))
        saving(num)
        continue
    elif keyboard_input == "3":
        num = int(input("请输入取款金额："))
        get_money(num)
        continue
    else:
        print("退出程序")
        break

8.函数多种传参方式

1）位置参数

  • 定义： 调用函数时根据函数定义的参数位置来传递参数

  • 示例：

    def user_info(name,age,gender):
    print(f"{name},{age},{gender}")


user_info("jojo", 20, "男")

    其中name=“jojo”，age=20，gender=”男“

    传递的参数和定义的参数的顺序及个数必须一致

2）关键字参数

  • 定义： 函数调用时通过“键=值”形式传递参数

  • 作用： 可以让函数更加清晰、容易使用，同时清楚了参数的顺序需求

  • 示例：

    def user_info(name, age, gender):
    print(f"{name},{age},{gender}")


user_info(name="jojo", age=20, gender="男")

    函数调用时，如果有位置参数，未知参数必须在关键字参数的前面，但关键字参数之间不存在先后顺序

3）缺省参数

  • 定义： 缺省参数，也叫默认参数，用于定义函数，为参数提供默认值，调用函数时可以不传该默认值的参数

  • 示例：

    def user_info(name, age, gender='男'):
    print(f"{name},{age},{gender}")


user_info(name="jojo", age=20)
user_info(name="jane", age=20, gender='女')

    # 输出结果
jojo,20,男
jane,20,女

4） 不定长参数

  • 定义： 不定长参数，也叫可变参数，用于不确定调用的时候回传递多少个参数的场景

  • 不定长参数的类型：

    • 位置传递

      • 示例：

        def user_info(*args):
    print(args)


user_info("tom")
user_info("tom",18)

        # 输出结果
('tom',)
('tom', 18)

      • **注意：**传进的所有参数都会被args变量收集，它会根据传进参数的位置合并为一个元组tuple，args是元组类型

    • 关键字传递

5. 数据容器（序列）的切片

  • 序列： 内容连续、有序，可使用下标索引的一类数据容器

    列表、元组、字符串均可以视为序列

  • 序列的常用操作——切片：

    • 切片： 从一个序列中取出一个子序列

    • 列表、元组、字符串均支持进行切片操作

    • 语法： 序列[起始下标：结束小标：步长]

      表示从序列中，从指定位置开始，依次取出元素，到指定位置结束，得到一个新序列，左闭右开

      • 起始下标表示从何处开始，可以留空，留空视作从头开始
      • 结束下标(不含)表示何处结束，可以留空，留空视作截取到结尾
      • 步长表示依次取元素的间隔：
        • 步长为1，表示一个个取元素
        • 步长为2，表示每次跳过1个元素
        • 步长为N，表示每次跳过N-1个元素
        • 步长为负数，表示反向取（起始下标和结束下标也要反向标记）

    • 此操作不会影响序列本身，而是会得到一个全新的序列

  • 示例：

    # 对list进行切片，从1开始，4结束，步长1
my_list = [0,1,2,3,4,5,6]
result1 = my_list[1:4]
print(result1)

# 对tuple进行切片，从头开始，到最后结束，步长1
my_tuple = (0,1,2,3,4,5,6)
result2 = my_tuple[:]
print(result2)

# 对str进行切片，从头开始，到最后结束，步长2
my_str = "01234567"
result3 = my_str[::2]
print(result3)

# 对str进行切片，从头开始，到最后结束，步长-1
my_str = "01234567"
result4 = my_str[::-1]  # 相当于对序列反转
print(result4)

# 对列表list进行切片，从3开始，到1结束，步长-1
my_list = [0,1,2,3,4,5,6]
result5 = my_list[3:1:-1]
print(result5)

# 对元组进行切片，从头开始，到尾结束，步长为-2
my_tuple = (0,1,2,3,4,5,6)
result6 = my_tuple[::-2]
print(result6)

    # 输出结果
[1, 2, 3]
(0, 1, 2, 3, 4, 5, 6)
0246
76543210
[3, 2]
(6, 4, 2, 0)

  • 试题：

    有字符串：“hello dlrowhello”，得到“world”

    my_str = "hello dlrowhello"
# 方式一：倒序字符串，切片取出

print(new_str1)

# 方式二：切片取出，然后倒序
new_str2 = my_str[6:11][::-1]
print(new_str2)

# 方式三：用split分隔“ ”，replace替换掉hello
new_str3 = my_str.split(" ")[1].replace("hello","")[::-1]
print(new_str3)

    # 输出结果
world 
world

① add（）

  • 语法： 集合.add（元素）

② remove（）

  • 功能： 将指定元素从集合内移除，集合本身被修改，移除了元素

  • 语法： 集合.remove（元素）

    my_set = {"hello","world"}
my_set.remove("hello")
print(my_set)

    # 输出结果
{'world'}

③ pop（）

  • 功能： 从集合中随机取出一个元素，会得到一个元素的结果，同时集合本身被修改，元素被移除

  • 语法： 集合.pop（）

  • 示例：

    my_set = {"hello","world"}

print(element)
print(my_set)

    # 输出结果
hello
{'world'}

④ clear（）

  • 功能： 清空集合，集合本身被清空

  • 语法： 集合.clear（）

  • 示例：

print(my_set)

    # 输出结果
set()

⑤ difference（）

  • 功能： 取出集合1和集合2的差集（集合1有而集合2没有）

  • 结果： 返回一个新集合，集合1和集合2不变

  • 格式： 集合1.difference（集合2）

  • 示例：

  • 功能： 对比集合1和集合2，在集合1内，删除和集合2相同的元素

  • 结果： 集合1被修改，集合2不变

  • 格式： 集合1.difference_update (集合2)

  • 示例：

{1, 5, 6}

⑦ union（）

  • 功能： 将集合1和集合2组合成新集合

  • 结果： 得到新集合，集合1和集合2不变

  • 格式： 集合1.union（集合2）

⑧ len（）

  • 功能： 统计集合元素数量

  • 格式： len（集合）

  • 示例：

    set1 = {1,2,3,4,5,6,1,2,3,4,5,6}
length = len(set1)
print(length)

    # 输出结果
6

    注意：定义集合时，会默认进行去重

⑨ 遍历

  • 集合不支持下标索引，不能用while 循环，但是可以用for进行遍历

  • 示例：

    set1 = {1, 2, 3, 4, 5, 6, 1, 2, 3, 4, 5, 6}
length = len(set1)
print(length,end='\n\n')

i = None
for i in set1:
    print(i, end=" ")

    # 输出结果
6

1 2 3 4 5 6

3）特点

  • 可以容纳多个数据
  • 可以容纳不同类型的数据
  • 数据是无需存储的（不支持下标索引）
  • 不允许重复数据存在
  • 可以修改
  • 支持for循环

7. dict 字典

1） 定义

  • 通过一个key值找到对应的数据

  • 字典的定义，同样使用{}，不过存储的元素时一个个的键值对

my_dict = {key:value ,key:value,……,key:value}
# 定义空字典
my_dict = {}	# 方式一
my_dict = dict()	# 方式二

  • 示例：

    # 定义字典

print(f"{my_dict2},{type(my_dict2)}")
print(f"{my_dict3},{type(my_dict3)}")
print(f"{my_dict4},{type(my_dict4)}")

    # 输出结果
{'jojo': 99, 'jane': 88, 'john': 77},<class 'dict'>
{},<class 'dict'>
{},<class 'dict'>
{'jojo': 88, 'john': 77},<class 'dict'>

  • 字典同集合一样，不可以使用下标索引，但是可以通过key值来取得对应的Value

  • 示例：

77

8.总结

  • 序列类型： 列表、元组、字符串

    非序列类型： 集合、字典

  • 序列类型支持重复元素，非序列类型不支持重复元素

    序列类型支持下标索引，非序列类型不支持下标索引

  • 列表、集合、字典支持修改

    元组、字符串不支持修改

  • ![屏幕截图 2024-08-06 194641](C:/Users/carp/OneDrive/桌面/Python/屏幕截图 2024-08-06 194641.png)

  • 各类数据容器的特点，以及应用场景：

    • 列表：一批数据，可修改、可重复的存储场景
    • 元组：一批数据，不可修改、可重复的存储场景
    • 字符串：一串字符串的存储场景
    • 集合：一批数据，去重存储场景
    • 字典：一批数据，可以用key检索value的场景

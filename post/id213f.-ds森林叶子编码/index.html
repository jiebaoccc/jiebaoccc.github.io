【id213】F. DS森林叶子编码

Oct 26, 2024
2 minute read

题目描述

给定一组森林，编写程序生成对应的二叉树，输出这颗二叉树叶结点对应的二进制编码.规定二叉树的左边由0表示，二叉树的右边由1表示。

输入

N B 表示N个树，每结点最多B个分支

第2行至第N+1行，每个树的先序遍历

输出

每行表示一个叶结点对应的二进制编码.

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例：

3 3
A B 0 0 0 C 0 0 0 D 0 0 0
E F 0 0 0 0 0
G H 0 0 0 I J 0 0 0 0 0 0

输出样例：

0 1 1
1 0
1 1 0 1 0

解题思路：

首先将每一棵树转换成二叉树，然后从右到左，将右边二叉树的节点作为左边二叉树的右子树，最后对其进行先序遍历，如果朝右遍历则容器中插入1，左边则插入0

将树转换成二叉树：

首先创建根节点数组，然后依次将节点传入函数，如果为3叉树，则对每个节点进行3次循环，接受三个节点，如果节点不为空，则对该节点再调用此函数，调用完之后需要将根节点进行更新；同时，每个节点的左节点是3个节点中的首个非空节点

创建完二叉树后，将右边二叉树的节点作为左边二叉树的右子树

最后对其进行先序遍历，如果该节点没有左右子树，说明为叶子节点，则输出其容器，反之继续遍历非空子树

具体代码：

#include<bits/stdc++.h>
using namespace std;

class Tree
{
public:
	char data;
	Tree* left;
	Tree* right;
	Tree() {}
	Tree(char d) :data(d) {}
};

void Create_tree(Tree* root,int b)
{
	Tree* t = root;
	char ch,ch1;
	int tag = 1;
	for (int i = 0; i < b; i++)
	{
		ch = getchar();
		ch1 = getchar();
		if (ch != '0' && tag)
		{
			root->left = new Tree(ch);
			Create_tree(root->left, b);
			root = root->left;
			tag = 0;
		}
		else if(ch != '0' && !tag)
		{
			root->right = new Tree(ch);
			Create_tree(root->right, b);
			root = root->right;
		}
	}
}

void preorder(Tree*& t,vector<int> vec)
{
	if (!t->left && !t->right)
	{
		for (int i = 0; i < vec.size(); i++)
		{
			cout << vec[i];
			if (i < vec.size() - 1)
				cout << " ";
		}
		cout << endl;
		return;
	}

	vector<int> tem;
	tem.assign(vec.begin(),vec.end());
	if (t->left)
	{
		tem.push_back(0);
		preorder(t->left,tem);
	}

	tem.assign(vec.begin(), vec.end());
	if (t->right)
	{
		tem.push_back(1);
		preorder(t->right, tem);
	}
	return;
}

int main()
{
	int n, b;
	cin >> n >> b;
	getchar();
	vector<Tree*> t(n);
	for (int i = 0; i < n; i++)
	{
		char ch;
		cin >> ch;
		getchar();
		t[i] = new Tree(ch);

		Create_tree(t[i],b);
	}

	for (int i = n - 1; i > 0; i--)
	{
		t[i - 1]->right = t[i];
	}
	vector<int> vec;
	preorder(t[0],vec);

	return 0;
}

【id200】E. 先序+中序还原二叉树

Oct 26, 2024
1 minute read

题目描述

给定一棵二叉树的先序遍历序列和中序遍历序列，要求计算该二叉树的高度。

输入

输入首先给出正整数N（≤50），为树中结点总数。下面两行先后给出先序和中序遍历序列，均是长度为N的不包含重复英文字母（区别大小写）的字符串。

输出

输出为一个整数，即该二叉树的高度。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例：

9
ABDFGHIEC
FDHGIBEAC

输出样例：

5

解题思路：

首先读取先序和中续遍历，

因为先序遍历中第一个数永远为根节点，而中序遍历的根节点的左边为左子树的遍历，右边有右子树的遍历

所以运用动态规划，最大深度为左子树的深度与右子树深度的最大值

而左子树的先序遍历字符串为原先序字符串删除0下标的字符，然后从下标0开始，到左子树节点数量,右子树同理，如此递归

同时初始化h=1，当后序遍历的字符串的size=0，则返回0

backorder_left.assign(backorder_data.begin(), backorder_data.begin() + index);
backorder_right.assign(backorder_data.begin() + index + 1, backorder_data.end());

preorder_left.assign(preorder_data.begin(), preorder_data.end() - backorder_right.size());
preorder_right.assign(preorder_data.end() - backorder_right.size(), preorder_data.end());

具体代码：

#include<bits/stdc++.h>
#include<unordered_map>
using namespace std;

unordered_map<char, int> has;

int GetHeight(vector<char> preorder_data, vector<char> backorder_data)
{
	if (backorder_data.size() == 0)
		return 0;

	for (int i = 0; i < backorder_data.size(); i++)
	{
		has[backorder_data[i]] = i;
	}

	char data = preorder_data[0];
	preorder_data.erase(preorder_data.begin());

	int h = 1;
	int index = has[data];

	vector<char> preorder_left,backorder_left;
	vector<char> preorder_right, backorder_right;

	backorder_left.assign(backorder_data.begin(), backorder_data.begin() + index);
	backorder_right.assign(backorder_data.begin() + index + 1, backorder_data.end());

	preorder_left.assign(preorder_data.begin(), preorder_data.end() - backorder_right.size());
	preorder_right.assign(preorder_data.end() - backorder_right.size(), preorder_data.end());

	int h1 = 0, h2 = 0;
	h1 = GetHeight(preorder_left, backorder_left);
	h2 = GetHeight(preorder_right, backorder_right);
	h += max(h1,h2);
	return h;
}

int main()
{
	int n;
	cin >> n;
	vector<char> preorder_data;
	vector<char> backorder_data;
	for (int i = 0; i < n; i++)
	{
		char ch;
		cin >> ch;
		preorder_data.push_back(ch);
	}

	for (int i = 0; i < n; i++)
	{
		char ch;
		cin >> ch;
		backorder_data.push_back(ch);
	}

	int h = GetHeight(preorder_data, backorder_data);
	cout << h << endl;
	return 0;
}

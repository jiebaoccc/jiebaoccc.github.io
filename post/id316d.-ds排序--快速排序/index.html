【id316】D. DS排序--快速排序

Dec 12, 2024
2 minute read

题目描述

给出一个数据序列，使用快速排序算法进行从小到大的排序

–程序要求–

若使用C++只能include一个头文件iostream；若使用C语言只能include一个头文件stdio

程序中若include多过一个头文件，不看代码，作0分处理

不允许使用第三方对象或函数实现本题的要求

输入

第一行输入t，表示有t个测试示例 第二行输入n，表示第一个示例有n个数据 第三行输入n个数据，都是正整数，数据之间用空格隔开 以此类推

输出

每组测试数据，输出每趟快排的结果，即每次排好一个数字结果（长度为1的子序列，不用排，不用输出）。不同测试数据间用空行分隔。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中

解题思路

快速排序，运用递归，首先对原序列进行快排，获得枢纽位置，然后将枢纽左侧和右侧的序列进行快排，如此递归

快速排序的过程：

  1. 从high的位置开始，向前找到第一个数值小于Pivotkey的数，每找一次high- -，然后将该位置的数与low位置的数进行调换
  2. 再从low的位置开始，向后找到一个数值大于Pivotkey的数，每找一次low++，然后将其与high位置的数进行调换
  3. 重复步骤1,2，直到low = high
  4. Pivotkey的最终位置为low

具体代码

#include<iostream>
using namespace std;

void display(int* num, int n) {
	for (int i = 0; i < n; i++) {
		cout << num[i];
		if (i < n - 1) cout << " ";
		else cout << endl;	
	}
}

int Sort(int* num, int low, int high) {
	while (low < high) {
		while (num[high] >= num[low] && low < high) high--;
		swap(num[low],num[high]);

		while (num[low] <= num[high] && low < high) low++;
		swap(num[low], num[high]);
	}
	return low;
}

void Qsort(int* num, int n, int low, int high) {
	if (low < high) {
		int tag = Sort(num,low,high);
		display(num, n);
		Qsort(num, n,low,tag-1);
		Qsort(num, n, tag + 1, high);
	}
}

int main() {
	int t; cin >> t;
	while (t--) {
		int n; cin >> n;
		int* num = new int[n];
		for (int i = 0; i < n; i++) {
			cin >> num[i];
		}

		Qsort(num,n,0,n-1);
		cout << endl;
	}
	return 0;
}

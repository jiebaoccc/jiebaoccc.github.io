【id314】A. DS内排—直插排序

Dec 12, 2024
1 minute read

题目描述

给定一组数据，使用直插排序完成数据的升序排序。

–程序要求–

若使用C++只能include一个头文件iostream；若使用C语言只能include一个头文件stdio

程序中若include多过一个头文件，不看代码，作0分处理

不允许使用第三方对象或函数实现本题的要求

输入

数据个数n，n个数据

输出

直插排序的每一趟排序结果

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中

输入样例

7 34 23 677 2 1 453 3

输出样例

23 34 677 2 1 453 3
23 34 677 2 1 453 3
2 23 34 677 1 453 3
1 2 23 34 677 453 3
1 2 23 34 453 677 3
1 2 3 23 34 453 677

解题思路

当插入第 i (i >= 1)个元素时，前面r[0]-r[i-1]已经排好序了，令j=i-1

  • 将 r[i] 与 r[j] 进行比较，如果 r[i] < r[j] ，则 j- -
  • 不断重复上面的步骤，直到遇到 r[i] >= r[j]，则将 r[i] 插入到 r[j] 的前面，即将[r+1] - [i-1] 的元素往后移动一位
  • 从 i=1 开始，直到最后一位 i=n-1

具体代码

#include<iostream>
using namespace std;

int main() {
	int n; cin >> n;
	int* num = new int[n];
	for (int i = 0; i < n; i++)
		cin >> num[i];


	for (int i = 1; i < n; i++) {
		int j = i - 1,tem = num[i];
		while (num[i] < num[j])j--;
		
		for (int k = i - 1; k > j; k--)
			num[k + 1] = num[k];
		num[j + 1] = tem;

		for (int k = 0; k < n; k++)
		{
			cout << num[k];
			if (k < n - 1)
				cout << " ";
			else cout << endl;
		}
	}
	return 0;

}

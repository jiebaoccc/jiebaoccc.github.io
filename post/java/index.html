一、数据类型、运算符、表达式和语句

  1. 标识符：以字母、下划线、美元符开始，后接字母、下划线、美元符号、数字。长度不限

二、类与对象

类封装了一个对象类别的状态和方法，是用来创建对象的模板

类的基本格式：

class 类名{
    类体的内容;
}

//class 是用来定义类的关键字

类声明还可以包含以下信息：

  • 类的访问限制：public、private等
  • 通过关键字 extends 声明父类的名字，注意：Java中的类只能有一个直接父类
  • 通过关键字 implements 声明所要实现的接口的名字，一个类可以实现多个接口，多个接口的名字之间以逗号分隔
public class Sun extend Father imolements Playable,Workable{
    
}

// 声明了一个公共类Sun，它的父类是 Father，实现两个接口 Playbale 和Worable

习惯上，类名的第一个字母大写，当类名由几个单词复合而成时，每个单词的首字母大写，如“DateTime”

类体中有两种类型的成员：

  • 类中的成员变量，也称之为域，他们用来刻画对象的内部状态
  • 类中的方法，用来刻画对象堆堆袜提供的行为，如构造方法

其中成员变量，即域，其声明顺序为：

  • 零个或多个修饰符，如public修饰符用来表明一个域是公共的
  • 域的数据类型
  • 域的名称

成员方法的顺序：

  • 方法的访问限制符和其他的修饰符，如public、private、static
  • 方法的返回类型
  • 方法的名称

在类的定义中方法是可以重载的

构造方法，例子如下：

package helloworld;

public class Cuboid {
    int height = 20;
    int length =30;
    int depth=10;

    public Cuboid(){}
    public Cuboid(int h){height = h;}
    public Cuboid(int h,int l){height=h;length=l;}
    public Cuboid(int h,int l,int d){height=h;length=l;depth=d;}
    public int calVolume(){return height*length*depth;}

    public static void main(String[] args){
        Cuboid cb=new Cuboid(10,15);
        cb.depth=5;
        System.out.printf("Volume of the Cuboid:%d\n",cb.calVolume());
    }
}

域、成员变量：

  • 用关键字 static 修饰的成员变量称之为静态变量，也即类变量

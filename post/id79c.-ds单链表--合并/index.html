【id79】C. DS单链表--合并

Sep 12, 2024
1 minute read

题目描述

​ 假定两个单链表是递增有序，定义并实现以下函数，完成两个单链表的合并，继续保持递增有序

​ int LL_merge(ListNode *La, ListNode *Lb)

​ 题目要求：必须使用指针表示next域，没有表示出链表结点结构的扣一半分！！

输入

​ 第1行先输入n表示有n个数据，接着输入n个数据

​ 第2行先输入m表示有M个数据，接着输入m个数据

输出

​ 输出合并后的单链表数据，数据之间用空格隔开

IO模式

​ 本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例1：

3 11 33 55
4 22 44 66 88

输出样例1:

11 22 33 44 55 66 88

代码：

#include<iostream>
using namespace std;

class ListNode {
public:
	int val;
	ListNode* next=nullptr;
	
};

void display(ListNode* t)
{
	ListNode* p = t->next;
	while (p != nullptr)
	{
		cout << p->val << " ";
		p = p->next;
	}
	cout << endl;
}

ListNode* LL_merge(ListNode* La, ListNode* Lb)
{
	ListNode* p = La;
	ListNode* m = La->next;
	ListNode* n = Lb->next;

	while (m && n)
	{
		if (m->val < n->val)
		{
			p->next = m;
			m = m->next;
			p = p->next;
		}
		else
		{
			p->next = n;
			n = n->next;
			p = p->next;
		}
	}

	p->next = !m ? n : m;
	
	return La;
}

void insert(ListNode*& l,int n)
{
	ListNode* p = l;
	while (n--)
	{
		int num;
		cin >> num;
		ListNode* t = new ListNode;
		t->val = num;
		p->next = t;
		p = t;
	}
}



int main()
{
	int n;
	ListNode* l1 = new ListNode;
	ListNode* l2 = new ListNode;

	cin >> n;
	insert(l1,n);

	cin >> n;
	insert(l2, n);
	
	ListNode* t = LL_merge(l1, l2);
	display(t);
}

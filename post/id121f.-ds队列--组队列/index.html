【id121】F. DS队列--组队列

Oct 02, 2024
2 minute read

题目描述

组队列是队列结构中一种常见的队列结构，在很多地方有着广泛应用。组队列是是指队列内的元素分组聚集在一起。组队列包含两种命令：

1、 ENQUEUE，表示当有新的元素进入队列，首先会检索是否有同一组的元素已经存在，如果有，则新元素排在同组的最后，如果没有则插入队列末尾。

2、 DEQUEUE，表示队列头元素出队

3、 STOP，停止操作

输入

第1行输入一个t(t<=10)，表示1个队列中有多少个组

第2行输入一个第1组的元素个数和数值

第3行输入一个第2组的元素个数和数值

以此类推输入完t组以定义同组元素之后，开始输入多个操作命令(<200)，对空的组队列进行操作，例如输入ENQUEUE 100，表示把元素100插入队列

输出

DEQUEUE出队的元素

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例1：

2
3 101 102 103
3 201 202 203
ENQUEUE 101
ENQUEUE 201
ENQUEUE 102
ENQUEUE 202
ENQUEUE 103
ENQUEUE 203
DEQUEUE
DEQUEUE
DEQUEUE
STOP

输出样例1：

101 102 103

输入样例2：

3
3 101 102 103
3 201 202 203
3 301 302 303
ENQUEUE 201
ENQUEUE 301
ENQUEUE 102
DEQUEUE
DEQUEUE
DEQUEUE
ENQUEUE 101
ENQUEUE 203
ENQUEUE 302
ENQUEUE 301
DEQUEUE
DEQUEUE
DEQUEUE
STOP

输出样例2：

201 301 102 101 203 302

解题思路：

首先定义一个unordered_map，用于记录每个数的组序号

接着依次输入每个数，以及绑定其序号

定义一个容器vec，以及一个队列que，容器用于存放存入的数，队列用于存放弹出的数

当输入ENQUEUE指令时，判断vec是否为空，若为空则直接插入，否则遍历容器中与自己同组的数据，然后将其插入到同组的最后，若没有找到同组的，则插入到最后

当输入DEQUEUE指令时，将vec的头部数据存入队列que并删除

最后将队列que中的数据输出并弹出

具体代码：

#include<iostream>
#include<vector>
#include<queue>
#include<unordered_map>
using namespace std;

unordered_map<int, int> group;

int main()
{
	int t;
	cin >> t;
	int* len = new int[t];
	vector<int> vec;
	queue<int> que;
	for (int i = 0; i < t; i++)
	{
		int n,num;
		cin >> n;
		len[i] = n;
		for (int j = 0; j < n; j++)
		{
			cin >> num;
			group[num] = i;
		}
	}

	string s;
	while (1)
	{
		cin >> s;
		if (s == "STOP")
			break;
		else if (s == "ENQUEUE")
		{
			int num;
			cin >> num;
			if (vec.empty())
				vec.push_back(num);
			else
			{
				int j = -1;
				for (int i = 0; i < vec.size(); i++)
				{
					if (group[num] == group[vec[i]])
						j = i;
				}
				if (j == -1)
					vec.push_back(num);
				else
					vec.insert(vec.begin()+j+1,num);
			}
		}
		else
		{
			que.push(vec.front());
			vec.erase(vec.begin());
		}
	}

	int tag=0;
	while (!que.empty())
	{
		if (tag)
			cout << " ";
		cout << que.front();
		que.pop();
		tag = 1;
	}
	cout << endl;
	return 0;
}

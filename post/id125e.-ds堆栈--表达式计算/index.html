【id:125】E. DS堆栈--表达式计算

Sep 19, 2024
3 minute read

题目描述

计算一个表达式的运算结果

使用C++自带stack堆栈对象来实现

例如

\1. Push (OPTR, ‘#’);表示把字符#压入堆栈OPTR中，转换成c++代码就是OPTR.push(’#’);

\2. Pop(OPND, a); 表示弹出栈OPND的栈顶元素，并把栈顶元素放入变量a中。因此改成c++代码是两个操作：

a = OPND.top(); OPND.pop();

\3. a = GetTop(OPND)表示获取栈OPND的栈顶元素，转成c++代码就是： a = OPND.top();

输入

第一个输入t，表示有t个实例

第二行起，每行输入一个表达式，每个表达式末尾带#表示结束

输入t行

输出

每行输出一个表达式的计算结果，计算结果用浮点数（含4位小数）的格式表示

用cout控制浮点数输出的小数位数，需要增加一个库文件，并使用fixed和setprecision函数，代码如下：

#include #include using namespace std;

int main()

{ double temp = 12.34

cout«fixed«setprecision(4)«temp«endl;

}

输出结果为12.3400

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中。

输入样例：

2
1+2*3-4/5#
(66+(((11+22)*2-33)/3+6)*2)-45.6789#

输出样例：

6.2000
54.3211

解题思路：

对于一个式子，如1+2*3-4/5，主体思路是将中缀表达式转换为后缀表达式，并通过后缀表达式进行计算

首先定义两个栈，一个存储数字 stack<double> num ，一个存储运算符 stack<double> op

从头到尾读取中缀表达式的每个对象，对不同对象按不同的情况处理。 对象分下列6种情况：

  1. 如果遇到空格，则认为是分隔符，不需处理，跳过
  2. 若遇到运算数，则存入num中
  3. 若是左括号，将其压入op中
  4. 若遇到右括号，表明括号内的中缀表达式已经扫描完毕，将op栈顶的运算符弹出并计算，直到遇到左括号（左括号也出栈，但是不做处理）
  5. 若遇到运算符，若该运算符的优先级大于栈顶运算符的优先级，将其压入op；若小于或等于，将栈顶运算符弹出并运算，再比较新的栈顶运算符，按同样的处理方法，直到该运算符大于栈顶运算符优先级为止，然后将该运算符压栈
  6. 若中缀表达式处理完毕，则将op中的运算符依次出栈并运算

连续运算符间的优先关系：

顺序 + - * / （ ） #
+ > > < < < > >
- > > < < < > >
* > > > > < > >
/ > > > > < > >
（ < < < < < = ×
） > > > > X > >
# < < < < < X =

分析：1+2*3-4/5

1入栈，+入栈，2入栈，因为*的优先级比+大，也入栈，3入栈

op num
* 3
+ 2
1

接着读取到 - ，此时 - 的优先级比 * 小，所以*出栈，并计算

3 * 2 = 6

所以栈更新为

op num
- 6
+ 1

接着4入栈，/入栈，5入栈

栈更新为

op num
/ 5
- 4
+ 6
1

最后开始计算，

5/4=0.8

6-0.8=5.2

1+5.2=6.2

所以最后得到1+2*3-4/5=6.2

具体代码示例

#include<iostream>
#include<stack>
#include<unordered_map>
#include<iomanip>
#include<math.h>
using namespace std;

char Prior[7][7] = {
	'>','>','<','<','<','>','>',
	'>','>','<','<','<','>','>',
	'>','>','>','>','<','>','>',
	'>','>','>','>','<','>','>',
	'<','<','<','<','<','=',' ',
	'>','>','>','>',' ','>','>',
	'<','<','<','<','<',' ','=' 
};

char opset[7] = { '+','-','*','/','(',')' };

int compare(char ch1, char ch2)
{
	int a, b;
	for (int i = 0; i < 7; i++)
	{
		if (opset[i] == ch1)
			a = i;
		if (opset[i] == ch2)
			b = i;
	}
	if (Prior[b][a] == '<')
		return 1;
	else return 0;
}

int main()
{
	int t;
	cin >> t;
	char ch = getchar();
	while (t--)
	{
		stack<char> op;
		stack<double> num;
		int tag=1,i=1,key=0;
		double n = 0;

		while ((ch = getchar()) != '#')
		{
			if (ch == '.')
			{
				tag = 0;
				continue;
			}
			if (ch >= '0' && ch <= '9')
			{
				if (tag)
				{
					n = n * 10 + (ch - '0');
					key = 1;
				}
				else
				{
					n += (ch - '0') * pow(0.1, i++);
					key = 1;
				}
			}
			else
			{
				if (key)
				{
					num.push(n);
					tag = i = 1;
					n = 0;
					key = 0;
				}

				if (ch == ')')
				{
					while (op.top()!='(')
					{
						double a, b;
						char temp = op.top();
						op.pop();
						switch (temp)
						{
						case'*':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b *= a;
							num.push(b);
							break;
						case'/':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b /= a;
							num.push(b);
							break;
						case'+':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b += a;
							num.push(b);
							break;
						case'-':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b -= a;
							num.push(b);
							break;
						}
					}
					op.pop();
				}
				else if (!op.empty())
				{
					if (compare(ch, op.top()))
					{
						op.push(ch);
					}
					else
					{
						char temp = op.top();
						op.pop();
						op.push(ch);

						double a, b;
						switch (temp)
						{
						case'*':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b *= a;
							num.push(b);
							break;
						case'/':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b /= a;
							num.push(b);
							break;
						case'+':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b += a;
							num.push(b);
							break;
						case'-':
							a = num.top();
							num.pop();
							b = num.top();
							num.pop();
							b -= a;
							num.push(b);
							break;
						}
					}
				}
				else
					op.push(ch);
			}
		}

		num.push(n);
		while (!op.empty())
		{
			double a, b;
			char temp = op.top();
			op.pop();
			switch (temp)
			{
			case'*':
				a = num.top();
				num.pop();
				b = num.top();
				num.pop();
				b *= a;
				num.push(b);
				break;
			case'/':
				a = num.top();
				num.pop();
				b = num.top();
				num.pop();
				b /= a;
				num.push(b);
				break;
			case'+':
				a = num.top();
				num.pop();
				b = num.top();
				num.pop();
				b += a;
				num.push(b);
				break;
			case'-':
				a = num.top();
				num.pop();
				b = num.top();
				num.pop();
				b -= a;
				num.push(b);
				break;
			}
		}
		cout <<fixed<<setprecision(4)<< num.top() << endl;
		ch = getchar();
	}
}

【id268】E. 旅游规划

Nov 21, 2024
1 minute read

题目描述

有了一张自驾旅游路线图，你会知道城市间的高速公路长度、以及该公路要收取的过路费。现在需要你写一个程序，帮助前来咨询的游客找一条出发地和目的地之间的最短路径。如果有若干条路径都是最短的，那么需要输出最便宜的一条路径。

输入

输入说明：输入数据的第1行给出4个正整数N、M、S、D，其中N（2≤N≤500）是城市的个数，顺便假设城市的编号为0~(N−1)；M是高速公路的条数；S是出发地的城市编号；D是目的地的城市编号。随后的M行中，每行给出一条高速公路的信息，分别是：城市1、城市2、高速公路长度、收费额，中间用空格分开，数字均为整数且不超过500。输入保证解的存在。

输出

在一行里输出路径的长度和收费总额，数字间以空格分隔，输出结尾不能有多余空格。

IO模式

本题IO模式为标准输入/输出(Standard IO)，你需要从标准输入流中读入数据，并将答案输出至标准输出流中

输入样例1

4 5 0 3
0 1 1 20
1 3 2 30
0 3 4 10
0 2 2 20
2 3 1 20

输出样例1

3 40

解题思路

通过Floyd算法，将城市i到城市j的最短路径求出来，其中如果路径相同，则判断花费，选择花费小的

最后输出path[s][d]、costs[s][d]

具体代码

#include<bits/stdc++.h>
using namespace std;

void Floyd(vector<vector<int>>& arr, vector<vector<int>>& costs,int m)
{
	for (int k = 0; k < m; k++)
	{
		for (int i = 0; i < m; i++)
		{
			for (int j = 0; j < m; j++)
			{
				if (arr[i][j] > arr[i][k] + arr[k][j])
				{
					arr[i][j] = arr[i][k] + arr[k][j];
					costs[i][j] = costs[i][k] + costs[k][j];
				}
				else if (arr[i][j] == arr[i][k] + arr[k][j])
				{
					if (costs[i][j] > costs[i][k] + costs[k][j])
					{
						arr[i][j] = arr[i][k] + arr[k][j];
						costs[i][j] = costs[i][k] + costs[k][j];
					}
				}
			}
		}
	}
}

int main()
{
	int n, m, s, d;
	cin >> n >> m >> s >> d;

	vector<vector<int>> path(m,vector<int>(m,INT_MAX/8));
	vector<vector<int>> costs(m, vector<int>(m, INT_MAX/8));

	for (int i = 0; i < m; i++)
	{
		int x, y, dis, cost;
		cin >> x >> y >> dis >> cost;
		path[x][y] = dis;
		costs[x][y] = cost;
	}

	Floyd(path,costs,m);

	cout << path[s][d] <<" " << costs[s][d] << endl;
}
